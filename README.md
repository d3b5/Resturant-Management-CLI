# 🍽️ Restaurant Management CLI

A simple command-line based Restaurant Management System built using Object-Oriented Programming principles.  
This project simulates basic restaurant operations such as managing menu items, taking orders, and handling interactions through a CLI interface.

---

## 🚀 Features

- 📋 View and manage menu items
- 🧾 Place and process customer orders
- 💰 Basic billing calculation
- 🧑‍🍳 Structured using OOP concepts
- 🖥️ Interactive command-line interface
